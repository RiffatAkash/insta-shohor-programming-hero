How Does Javascript Work??

Javascript Engine Program Can go by different names in different browsers but they're just names for the javascript Engine of that browser for example Firefox has SpiderMonkey, google chrome has the v8 Engine, Safari has nitro and Ie has chakra. Javascript Engine's main job is to take the javascript syntax that's in human-readable form and convert it ultimately into machine code so that our computer can understand our instructions. This is why we called it client-side because the files are being downloaded onto the client's computer and processed onto the client's computer..

Local Storage Vs Session Storage

Session Storage and Local Storage are very similar in how they interact and are only different in a few instances while cookies are almost completely different than the other two and also quite a bit older than the other two. Local Storage and Session Storage can hold 10 megabits and 5 megabits respectively. session storage is only available in the single tab that you have open that you set it in so it won't be available if they open another tab and go to your website. This is the case where local and session storage is different from each other.
